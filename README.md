# Monkey-study

> 深入研究 [zhuweileo/Monkey](https://github.com/zhuweileo/Monkey) 的技术实现

## 项目简介

Monkey 是一个浏览器自动化工具，本项目通过源码阅读和实验，深入理解其架构设计与核心实现。

## 目录结构

```
Monkey-study/
├── Monkey/          # Monkey 源码
├── notes/           # 学习笔记
└── scripts/         # 实验脚本
```

## 学习内容

- 浏览器自动化原理
- 核心架构设计分析
- 可复用模式提取
